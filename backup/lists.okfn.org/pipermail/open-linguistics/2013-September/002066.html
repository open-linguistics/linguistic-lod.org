Hello,

  okfnpad.org will be deprecated on the 15th of October 2013, in favor of a
new version of etherpad, which is currently available at
http://new.okfnpad.org.
This upgrade/move is necessary since the older version is no longer
maintained and has security issues.

Please note the newer version does not support teampads/pro accounts, what
this means is that you will no longer be able to create accounts, under
which you can have multiple pads e.g <account>.okfnpad.org (subdomains,
private pads will not be possible).

 It is not possible for OKF to backup everyone's data, okfnpad users should
backup all pad contents before October 15th.

For the full termination notice :
http://new.okfnpad.org/p/etherpad-termination-notice

Some pointers on exporting teampad content,

To list all teampads within your account:
http://<account>.okfnpad.org/ep/padlist/all-pads?undefined

To export teampads in a zip file:
http://<account>.okfnpad.org/ep/padlist/all-pads.zip

Note: to successfully export all teampads as a .zip, you must ensure they
all have unique names, else you might end up with an error like 'Oops! A
server error occured. It's been logged.'

Fetch pads individually as a txt file, you'll need to know the pad number
(can be figured by hovering the all-pads list) :
 http://<account>.okfnpad.org/ep/pad/export/<pad number>/latest?format=txt
