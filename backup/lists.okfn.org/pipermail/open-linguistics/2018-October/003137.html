The Sixteenth Extended Semantic Web Conference (ESWC 2019)
2nd June - 6th June 2019, Portorož, Slovenia
 
Web Page: https://2019.eswc-conferences.org <https://2019.eswc-conferences.org/>   
Twitter: @eswc_conf 
Facebook: https://www.facebook.com/ESWCCONF <https://www.facebook.com/ESWCCONF>

CALL FOR TUTORIALS
	ESWC 2019 invites tutorials for people new to the Semantic Web, researchers and practitioners that wish to learn new technologies, users of Semantic Web technologies, and representatives of government and funding agencies as well as potential private investors in Semantic Web technologies.
We are open to all tutorials, with a particular focus on:
	- Introduction to new semantic technologies, methods, techniques, and trends
	- Applications of semantic technologies in specific domains
	- Techniques from other fields relevant for Semantic Web research

CALL FOR WORKSHOPS 
	Co-located workshops at the ESWC 2019 are an essential meeting point for discussing ongoing work and the latest research ideas. We especially encourage proposals on:
	- Fundamental technical and theoretical problems
	- Applications of Semantic Web technologies
	- Key enabling technologies and their adaptation to the needs of the Semantic Web
        - Techniques from other fields relevant for Semantic Web research
	- New emerging topics and areas

Important Dates
	* Proposals due: November 23rd,  2018 - 23:59 Hawaii Time
	* Notification of acceptance: December 3rd,  2018 - 23:59 Hawaii Time
	* Website due: December 20th, 2018 - 23:59 Hawaii Time
	* Camera-ready proceedings due:  May 3rd, 2019 - 23:59 Hawaii Time
	* Workshop/Tutorial days: 2nd and 3rd of June, 2019


	Suggested Timeline for Workshop Papers
	* Submission deadline: Friday March 1st, 2019
	* Notifications: Friday March 29th, 2019
	* Camera-ready version: Friday April 12th, 2019

Submission via easychair https://easychair.org/conferences/?conf=eswc2019workshoptuto <https://easychair.org/conferences/?conf=eswc2019workshoptuto>
