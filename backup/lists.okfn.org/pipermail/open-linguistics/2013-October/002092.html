We are happy to announce the release of UBY 0.4.0:

http://code.google.com/p/uby/

UBY is a linked lexical-semantic resource for natural language processing (NLP)
based on the ISO standard Lexical Markup Framework (LMF):

http://www.ukp.tu-darmstadt.de/uby/


New in the UBY release 0.4.0:

- upgrade to uimaFIT 2.0.0 and DKPro Core 1.5.0

- extension of the uima module: annotator that implement the SemanticTagProvider interface in DKPro Core based on UBY
