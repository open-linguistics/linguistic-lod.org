Hello all,

I have published a new resource, the ALPINO treebank
<http://www.let.rug.nl/vannoord/trees/> as
RDF. The resource is available from:

http://lexinfo.net/corpora/alpino/

In particular, I did this as I wanted to find-out how (and how easily) it
was to publish a linguistic linked data resource using off-the-shelf
technology. I therefore documented the process in the following posts:

http://maccaronic.wordpress.com/2012/08/29/lexical-linked-data-case-study-alpino-treebank/
http://maccaronic.wordpress.com/2012/08/30/lexical-linked-data-case-study-alpino-treebank-part-2/

Anyone I chose the ALPINO resource for this case study as it was already in
XML so easy to convert, open licensed GPLv2 and the current LLD cloud
doesn't have a corpus (AFAIK).

I hope this resource and the case study will be of interest and use to
members of this group

Regards,
John
