Dear all,

please find attached a list of submissions to our call for Multilingual 
Linked Open Data Semantic Web Journal Special Issue[1].
We are especially fond of the open and transparent reviewing process 
employed by the Semantic Web Journal [2], which entails, that you are 
able to see the publications as well as the reviews. The Semantic Web 
Journal encourages public reviews and comments. So please feel free to 
add your own reviews on any of the submissions.

Note that we have also updated the MLODE Workshop website, but there is 
still some content missing: [3]
