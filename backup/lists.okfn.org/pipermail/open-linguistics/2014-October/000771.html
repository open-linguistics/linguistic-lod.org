The International Conference on Computer Science, Computer Engineering, 
and Social Media (CSCESM2014)

Metropolitan College, Thessaloniki, Greece
December 12-14, 2014 | cscesm14 at sdiwc.net
http://sdiwc.net/conferences/cscesm2014/

All registered papers will be included in SDIWC Digital Library.
================================================================
The conference aims to enable researchers build connections between 
different digital applications. The CSCESM2014 welcomes submissions on 
any topic in the field of Computer Science, Computer Engineering, Social 
Media and any related topics. The event will be held over three days, 
with presentations delivered by researchers from the international 
community, including presentations from keynote speakers and 
state-of-the-art lectures.

RESEARCH TOPICS ARE NOT LIMITED TO:

- Access Controls
- Automated Software Engineering
- Bioinformatics and Scientific Computing
- Biometrics Technologies
- Cloud Computing
- Collaborative filtering
- Computational Intelligence
- Computer Animation
- Computer Architecture
- Computer Crime Prevention and Detection
- Computer Forensics
- Computer Modeling
- Computer Networks
- Computer Security
- Computer-aided Design
- Cryptography and Data Protection
- Data Compression
- Data Mining
- Digital Signal and Image Processing
- Digital System and Logic Design
- Distributed and Parallel Applications
- E-Learning
- E-Technology
- Expert Systems
- Forensics, Recognition Technologies and Applications
- Graphics and visualization
- Human-computer interaction
- Image / multimedia processing
- Information propagation and assimilation in social networks
- Internet and Web Applications
- Mobile Computing
- Multimedia Applications
- Multimedia Computing
- Natural language processing
- Network Security
- Network Security and Cryptography
- Qualitative and quantitative studies of social media
- Social innovation and effecting change
- Social Media Networking
- Social network analysis
- Social networking sites
- Social networks and online education
- Wireless Communications
- Wireless Sensor Network

Researchers are encouraged to submit their work electronically. All 
papers will be fully refereed by a minimum of two specialized referees. 
Before final acceptance, all referees comments must be considered.

IMPORTANT DATES: (Subject to change, visit the website for updates)

Submission Dates: Open from now until November 20, 2014
Notification of Acceptance: November 18, 2014 or 4 weeks from the 
submission date
Camera Ready Submission: December 02, 2014
Registration Deadline: December 02, 2014
Conference Dates: December 12-14, 2014
