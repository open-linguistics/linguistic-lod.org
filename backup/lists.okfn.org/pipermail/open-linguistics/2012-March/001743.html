Hi,

I'm not sure I've introduced myself on this list yet, so a quick hello
first - I'm Laura, a new(ish!) Community Coordinator at the OKFN. For those
of you I haven't yet met - hi! :)

I have been asked to oversee working groups generally, and am currently
checking that all groups are happy and functioning. If you have any
questions or are in need of help, do feel free to contact me at any time.

I can see this mailing list is well used, which is fantastic. The Open
Linguistics website (http://linguistics.okfn.org/) however is less active
at present. If you like, I would be very happy to deploy the new OKFN theme
there (see e.g. Open Economics <http://openeconomics.net/> and Open
Science<http://science.okfn.org/>for ideas of how it might look).

How you use the site is entirely up to you, and there is no pressure to
make any changes if you're currently happy. But if e.g. a nice a landing
page would be helpful, I can certainly help to get that up and running.

Let me know what you think!

All the best,
Laura
