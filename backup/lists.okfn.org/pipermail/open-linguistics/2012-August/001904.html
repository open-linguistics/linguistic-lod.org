Dear all,
one point I would like to clarify:
Martin Brümmer has the role of our new LLOD cloud image updater and 
stylist. The official LLOD cloud image is, as it always has been: 
http://linguistics.okfn.org/resources/llod/
For the next weeks until MLODE, while we discuss, debug and improve the 
datasets and links we will use 
https://docs.google.com/drawings/d/1LSSy516nMpbb-adXSBqpQ1D-aLhaTO0OeYVLfd0ErGc/edit?pli=1 
, as this is definitely easier to edit (lower barrier, so anyone can add 
datasets and *comments* with a right click, easily). Martin Brümmer will 
synchronize both versions from time to time and merge improvements into 
http://linguistics.okfn.org/resources/llod/

All the best,
Sebastian
