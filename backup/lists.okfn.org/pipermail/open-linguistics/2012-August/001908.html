Hi all,
CKAN has substantially changed since last July, when we developed the CKAN
validator for producing the 2011 version of the LOD cloud. Changes include
the URL (from ckan.net to thedatahub.org), the interface, available
functions, etc.

Unfortunately I do not have time to make the upgrade. But the code is
available from GitHub: https://github.com/anjeve/ckan-lod-validator

Anybody can send pull requests and we'd be glad to incorporate them.
