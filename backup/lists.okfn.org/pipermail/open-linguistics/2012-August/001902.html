Dear all,

to give an overview of the current state of the LLOD-Cloud and to keep 
track of the changes, I made a collaborative diagram. You can find it 
here: 
https://docs.google.com/drawings/d/1LSSy516nMpbb-adXSBqpQ1D-aLhaTO0OeYVLfd0ErGc/edit. 


I already included John McCrea's notes 
(http://wiki.okfn.org/Working_Groups/linguistics/Resources_in_the_cloud). We 
should try to keep the diagram up to date as some kind of roadmap. So if 
new datasets are accepted or errors are fixed, one should update the 
diagram.
Also note that I took the linking lines from the diagram at 
http://linguistics.okfn.org/resources/llod/ so the links between the 
datasets are still to be validated. Shortly before Sabre, I will use the 
Google document to make a (somewhat) nice-to-look-at diagram of the 
final state.

Regards,
Martin Brümmer
