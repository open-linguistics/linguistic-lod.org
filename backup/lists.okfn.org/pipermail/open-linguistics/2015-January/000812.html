Hi everyone,

The NewsReader project[1] is organising two hackathons this month, one in Amsterdam and one in London. During these hackathons, developers and researchers are invited to dig into data that was extracted from millions of news articles using an NLP & Semantic Web pipeline (you don’t need to know any Semantic Web technology to play with this, because we also have a webAPI). Attendance is free.

Amsterdam event (January 21): https://www.eventbrite.com/e/porsches-to-pizza-hack-6000000-automotive-news-articles-newsreader-tickets-14504369961
London event (January 30): http://www.eventbrite.com/e/porsches-to-pizza-hack-6000000-automotive-news-articles-newsreader-tickets-14458478699
