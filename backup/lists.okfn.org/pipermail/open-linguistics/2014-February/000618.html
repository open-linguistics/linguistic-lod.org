On February 27, 4:00 PM CET the LOD2 project (http://lod2.eu 
<http://lod2.eu>) will offer a free one-hour webinar on Natural Language 
Processing Interchange Format (NIF).

NIF is based on a Linked Data enabled URI scheme for identifying 
elements in (hyper-)texts and an ontology for describing common NLP 
terms and concepts. In contrast to more centralized solutions such as 
UIMA and GATE, NIF enables the creation of heterogeneous, distributed 
and loosely coupled NLP applications, which use the Web as an 
integration platform. We present several use cases of the second version 
of the NIF specification (NIF 2.0) and the result of a developer study.
